# Sistem Informasi & Manajemen Kas Masjid (SIKAM)

Aplikasi web modern untuk transparansi keuangan dan manajemen kegiatan masjid. Dibangun dengan fokus pada performa, kemudahan penggunaan, dan desain yang bersih (*Enterprise Clean Design*).

Proyek ini dikembangkan untuk **Masjid Manarul Islam Bangil**.

## ğŸš€ Panduan Instalasi (Local Development)

Ikuti langkah ini untuk menjalankan proyek di komputer Anda.

### 1. Clone Repository
```bash
git clone https://github.com/SahrulRamadhanHardiansyah/masjid-app
cd masjid-app

```

### 2. Install Dependencies

```bash
npm install

```

### 3. Konfigurasi Environment Variable

Buat file `.env.local` di root folder dan isi dengan kredensial Supabase Anda:

```env
NEXT_PUBLIC_SUPABASE_URL=[https://your-project-id.supabase.co]
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here

```

### 4. Setup Database (Supabase SQL)

Buka **SQL Editor** di dashboard Supabase Anda dan jalankan script berikut untuk membuat tabel yang dibutuhkan:

```sql
-- 1. Tabel Kategori Transaksi
create table transaction_categories (
  id bigint generated by default as identity primary key,
  name text not null,
  type text check (type in ('income', 'expense')),
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- 2. Tabel Transaksi
create table transactions (
  id bigint generated by default as identity primary key,
  date date not null,
  amount numeric not null,
  description text,
  type text check (type in ('income', 'expense')),
  category_id bigint references transaction_categories(id),
  created_by uuid references auth.users(id),
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- 3. Tabel Agenda (Events)
create table events (
  id bigint generated by default as identity primary key,
  title text not null,
  short_description text,
  location text,
  pic_name text,
  start_datetime timestamp with time zone not null,
  end_datetime timestamp with time zone,
  created_by uuid references auth.users(id),
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- (Opsional) Insert Data Dummy Kategori Awal
insert into transaction_categories (name, type) values
('Infak Jumat', 'income'),
('Wakaf Pembangunan', 'income'),
('Listrik & Air', 'expense'),
('Honor Imam/Khatib', 'expense'),
('Kebersihan', 'expense');

```

### 5. Jalankan Server

```bash
npm run dev

```

Buka browser dan akses `http://localhost:3000`.
